# 聊天输入与触发面板

## 工具条

仅保留 **添加附件**。其余用字符触发：

| 字符 | 面板 |
|------|------|
| `/` | 对话模式（命令） |
| `@` | 提及附件 / 知识库 |
| `$` 或 `¥` | 技能 |
| `#` | 插入其它会话 |

## 附件

- 只显示文件名 chip  
- 支持整页拖放 + 全屏「松开以添加附件」  
- 若路径被粘贴进输入框，自动转成附件并清除路径文本  

## 上下文

``ContextUsageBar`` / ``ContextEstimator``：按字符估算占用，分桶展示消息、知识库、附件、技能、系统提示、插入会话。
