# 产品决策（长久记忆）

> 已确认的产品约束。后续改动前请先对照本页。

## 平台与品牌

- 产品名：**黑妞短剧**
- Bundle ID：`cn.codable.heiniu`
- 仅 **macOS 15+**，SwiftUI，中文界面
- 开源协议：MIT

## 服务商

- LLM：OpenAI 兼容（Chat Completions / Responses）+ Anthropic  
- 生图 / 生视频：可配置 **多家** 服务商  
- 服务商列表默认 **折叠**；通过右侧菜单「编辑」展开  

## 黑妞

- 类似 Gemini Gem / Custom GPT 的自定义角色  
- 编辑页布局：**左侧分类导航 + 右侧详情**  
  - 模型设置 / 提示词 / 知识库 / MCP 策略 / 常用短语 / 基本资料  
- 知识库在编辑页配置，**不在聊天工具条放入口**  

## 命令 vs 技能 vs 插件

| 概念 | 触发 | 管理位置 | 说明 |
|------|------|----------|------|
| 对话模式（命令） | `/goal` `/plan`… | 系统内置 | 工作方式，不是技能 |
| 技能 | `$outline`… | 配置 → 技能 | 可复用能力包 |
| 插件 | — | 配置 → 技能 → 插件 | 技能容器，可启用/禁用 |

- 归属插件的技能 **只在插件页展示**，不出现在独立技能列表，避免冲突  
- MCP：全局清单在「配置 → MCP」；黑妞侧仅选 **禁用 / 自动 / 手动**  

## 聊天输入

- 工具条只保留 **附件**  
- `/` 命令 · `@` 提及 · `$`/`¥` 技能 · `#` 插入会话  
- 拖放目标为 **整个聊天界面**，全屏提示「松开以添加附件」  
- 附件只显示 **文件名 chip**，不显示绝对路径
